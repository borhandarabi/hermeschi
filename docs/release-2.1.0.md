# Hermes-Workspace نسخهٔ ۲.۱.۰ — Swarm

Hermes-Workspace ۲.۱.۰ **Swarm** را معرفی می‌کند، یک لایهٔ هماهنگی چندعاملی داخلی برای اجرای یک ClaudeAgent اصلی در کنار عامل‌های worker پایدار.

## نکات برجسته

- **حالت Swarm**
  - مسیریابی کار از یک ClaudeAgent اصلی به یک swarm زنده از workerها
  - workerهای پایدار با پشتیبانی tmux
  - لایه‌های dispatch و هماهنگی آگاه از نقش

- **جریان Board، گزارش‌ها و inbox**
  - پشتیبانی از Board / Kanban برای کار swarm
  - مسیریابی گزارش‌ها و checkpoint
  - مدیریت به‌سبک inbox برای کارهای مسدودشده و آماده بازبینی

- **مسیریابی Orchestrator**
  - checkpointهای worker اول از طریق orchestrator مسیریابی می‌شوند
  - جریان بازبینی‌کنندهٔ پاک‌تر و کنترل بهتر بر escalation

- **بهبود قابلیت اطمینان**
  - استریم‌های SSE گفتگوی طولانی‌مدت پنجره‌های پردازش خاموش عامل را تحمل می‌کنند
  - سیم‌کشی بنر تأیید اصلاح شد تا تأییدهای ابزار دوباره قابل‌مشاهده باشند
  - نشست‌های portable فقط‌محلی به‌درستی قابل حذف هستند
  - fallback داشبورد برای جریان‌های ایجاد/به‌روزرسانی/fork نشست افزوده شد
  - وصله‌های قابلیت اطمینان فضای کار حفظ شدند

- **اصلاحات مسیر Claude + محیط**
  - مدیریت رسمی root Claude
  - بهبود مدیریت home/env برای profileها و ذخیره‌سازی اجرا

- **مستندات و امنیت**
  - مستندات Swarm افزوده شد
  - مستندات fallback Docker Skills Hub شفاف‌سازی شد
  - مسیر افشای SECURITY به‌روزرسانی شد
